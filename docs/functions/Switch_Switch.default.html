  • default<T>(props): ReactElement | null

  • A declarative switch-case component for conditional rendering in React. Renders content based on matching a value against predefined cases.

    Type Parameters

    • T

      The type of the value to switch on (must be a valid object key)

    Parameters

    • props: SwitchProps<T>

      The props for the Switch component. See SwitchProps.

    Returns ReactElement | null

    ReactElement that renders either: - The matching case's content if a match is found - The 'Default' case content if provided and no match is found - null if no match is found and no Default case is provided

    Example

    // Basic usage with string value
    <Switch 
      value={status}
      cases={{
        loading: <LoadingSpinner />,
        error: <ErrorMessage />,
        success: <SuccessContent />,
        Default: <DefaultContent />
      }}
    />

    Example

    // Using with number value
    <Switch
      value={errorCode}
      cases={{
        404: <NotFound />,
        500: <ServerError />,
        Default: <UnknownError />
      }}
    />

    Example

    // Using with boolean value
    <Switch
      value={isAuthenticated}
      cases={{
        true: <AuthenticatedContent />,
        false: <LoginForm />,
      }}
    />

    Remarks

    • The component performs a strict equality check (===) for matching
    • Cases must be valid object keys (string, number, or symbol)
    • 'Default' is a special case that acts as a fallback
    • Performance is O(1) as it uses direct object property access
